Laboratori pratici di CTI difensiva.

Scenari guidati per applicare CTI, hunting e incident response senza contenuti offensivi.

Lab 1 · Phishing e furto credenziali

Scenario: un utente segnala una email con link a una falsa pagina di login. Poco dopo compaiono accessi da area geografica insolita.

Domande CTI

Quali IOC estrarre? Quali log verificare? È una campagna ampia o un caso isolato?

Telemetria

Email gateway, header, URL, IdP, MFA, mailbox audit, proxy, DNS.

Decisioni

Revoca sessioni, reset password, verifica regole mailbox, ricerca destinatari simili, avviso utenti.

Lab 2 · Ransomware

Scenario: endpoint con file cifrati, nota di riscatto, possibili accessi VPN anomali e traffico cloud in uscita prima dell’impatto.

Ipotesi

Accesso iniziale via credenziali, movimento laterale, staging dati, esfiltrazione, cifratura.

Log prioritari

VPN, AD, EDR, file server, proxy, DNS, DLP, backup, hypervisor.

Output CTI

MITRE mapping, blast radius, IOC, confidence, decisione di containment, limiti e fonti.

Lab 3 · CVE in CISA KEV

Scenario: una CVE appena entrata in KEV riguarda un prodotto esposto su internet usato dall’organizzazione.

Raccolta

CISA KEV, NVD, EPSS, inventory asset, vulnerability scanner, WAF, log applicativi.

Analisi

Asset esposto? Versione vulnerabile? Segni di exploitation? Compensating controls?

Decisione

Patch urgente, mitigazione temporanea, hunting retrospettivo, comunicazione rischio.

Lab 4 · Account cloud compromesso

Scenario: login impossibile, creazione app OAuth, download massivi e regola di inoltro email.

IOC/IOA

IP login, user agent, device, regola inbox, app OAuth, volume download.

Controlli

Conditional access, MFA phishing-resistant, revoke sessions, audit app, mailbox search.

Report

Account coinvolto, finestra temporale, dati accessibili, misure adottate, rischio residuo.

Griglia di valutazione laboratorio

Per ogni scenario completa questa scheda: evidenze osservate, fonti, tecniche MITRE, ipotesi, confidence, impatto, decisione, log mancanti, prossimi passi.
